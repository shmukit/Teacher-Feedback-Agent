# Contributing to Teacher Feedback Agent

Thank you for your interest in contributing! We welcome all contributions—whether it's fixing bugs, improving documentation, or adding new features (like new student personas or pedagogy dimensions).

## How to Contribute

1. **Fork the Repository:** Create your own fork of the project to your GitHub account.
2. **Clone Locally:** Clone the fork to your local machine.
3. **Create a Branch:** Create a feature branch (`git checkout -b feature/your-feature-name`).
4. **Make Changes:** Implement your feature or bug fix.
5. **Commit:** Commit your changes with clear, descriptive commit messages.
6. **Push:** Push the branch to your fork on GitHub.
7. **Pull Request:** Open a Pull Request against the `main` branch of this repository.
